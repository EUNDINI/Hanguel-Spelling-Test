## 주요 기능 및 구현 기술 🛠
* 반응형 웹사이트
  * HTML5, CSS, Javascript, Bootstrap을 사용
* 20개 퀴즈를 통해 점수에 따라 1~5단계로 결과 분류
* 정답 확인하기
* 공유하기 (카카오톡, 페이스북, 트위터, 링크 복사)
* 구글 애널리틱스를 통한 사용자 분석 
  * 하루 최대 1600명의 트래픽
